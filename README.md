# 🎮 Hangman Game in Python

A simple text-based **Hangman Game** built using Python. In this game, the computer randomly selects a word from a predefined list, and the player has to guess the word one letter at a time before running out of attempts.

---

## 📌 Project Overview

This project is designed for beginners to practice Python programming concepts such as loops, conditional statements, lists, strings, and the random module.

The player gets **6 incorrect attempts** to guess the hidden word. If all the letters are guessed correctly before the attempts are exhausted, the player wins.

---

## ✨ Features

- 🎲 Random word selection
- 🔤 Letter-by-letter guessing
- ✅ Correct and incorrect guess checking
- 🔁 Prevents duplicate guesses
- ❤️ 6 maximum incorrect attempts
- 💻 Simple console-based interface
- 🏆 Win/Lose result display

## 📖 How to Play

1. The computer randomly selects a word.
2. The hidden word is displayed using underscores (_).
3. Enter one letter at a time.
4. If the letter is correct, it is revealed.
5. If the letter is incorrect, one attempt is lost.
6. You have only **6 incorrect attempts**.
7. Guess the complete word before the attempts run out.
